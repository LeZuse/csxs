### Requirements

  * [Build and Install Node.js](https://github.com/joyent/node/wiki/Installation)
  * Install [NPM](http://npmjs.org/): `curl http://npmjs.org/install.sh | sh`
  * Install [csxs](https://github.com/diy/roto): `npm install -g csxs`
